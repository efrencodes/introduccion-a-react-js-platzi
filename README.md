### **Base React**

Project base with React.js

#### Stack

- Used technologies:
  - React js
  - SCSS
  - Eslint
  - Babel
  - Webpack
  - file-loader

#### Run the project

- Clone the repository.
- Run command `npm install` or `yarn install`.
- Open the project `http://localhost:8000`
