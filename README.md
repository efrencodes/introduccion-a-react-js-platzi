### ⚙️ Run the project

Para poder clonar y correr el proyecto, necesitas **GIT**  y **NodeJS** instalada en tu computadora.

```bash
  # Clonar el repositorio.
  $ git clone https://github.com/efrenmartinez/react-js-base.git

  # Ir a la carpeta.
  $ cd react-base

  # Instalar dependencias con npm
  $ npm install

  # Correr el proyecto
  $ npm run start
```
